# 🔐 Metasploitable2 Penetration Testing Lab

## 📌 Project Overview
This project demonstrates a hands-on penetration testing workflow carried out in a controlled lab environment using intentionally vulnerable systems.

The objective was to perform:
- Network scanning
- Vulnerability identification
- Exploitation
- Post-exploitation (Persistence)



## 🛠️ Tools & Technologies
- Nmap (Network Scanning)
- Metasploit Framework (Exploitation)
- Metasploitable2 (Target Machine)
- Kali Linux (Attacker Machine)



## 🔍 Phase 1: Network Scanning
A network scan was conducted to identify open ports and running services on the target system.

This phase helped in discovering potential entry points into the system.
